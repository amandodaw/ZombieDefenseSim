 # Game Design

 ## Genre
 2d Simulation/Colony Sym/Roguelike

 ## Player Fantasy
 The player decides what to build and who works in what task. Seeing all the creatures systems interactuate with each other. Each day there are more and stronger enemies, and the player's citicens become stronger and better at his skills. In the day, it's planning time. In the night, defense.

 ## Core Gameplay Loop

 Get basic resources

 Build basic food source

 Build basic defense

 Defende wave

 Look for more and better resources

 Buy better things

 Humans reproduce, population gets bigger

 More enemies come at night

 Each npc can have his own history thanks to systems
